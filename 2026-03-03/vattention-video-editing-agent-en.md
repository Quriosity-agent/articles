# Vattention: The Claude Code Moment for Video Editing?

> **TL;DR**: Video post-production AI Agent startup **Vattention** closes seed round (millions USD) from QingRui Capital, Baidu Ventures, Changlei Capital. Core thesis: video editing is moving from **PostPro 2.0 (non-linear)** to **PostPro 3.0 (exponential editing)**. Just as coding went from Copilot → Claude Code, Vattention builds a **Video-Editing Agent** — not generating video, but orchestrating raw footage into final cuts via Context Modeling.

---

## Key Insight: Editing Agent ≠ Generating Agent
```
Coding:  Code Generating (base model) vs Code-Editing Agent (Claude Code)
Video:   Video Generating (Sora) vs Video-Editing Agent (Vattention)
```
Base models generate; application-layer agents edit and orchestrate.

## Three Engines
- **MACE**: Training data — parse professional editing patterns
- **ACE**: Asset comprehension — deep feature extraction
- **PACE**: Core brain — real-time intent prediction + action guidance

## Why "Exponential Editing"?
More context accumulated → AI acceleration grows exponentially (not linearly). Same logic as Claude Code understanding your codebase better over time.

## Business Model: To P (Professional Consumer)
Target: tens of millions of professional creators globally. To P sits between To B (enterprise) and To C (consumer), capturing both.

## QCut Relevance
High overlap: both are video editing + AI Agent, both emphasize human-AI cocreation. Different entry: QCut = desktop timeline editor, Vattention = context modeling + predictive execution.
