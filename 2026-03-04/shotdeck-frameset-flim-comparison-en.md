# ShotDeck vs Frameset vs Flim.ai: Which Visual Reference Stack Should Creators Use?

> **TL;DR**: All three tools solve the same core problem: turning vague visual taste into clear creative direction. ShotDeck is strongest for deep cinematic reference search, Frameset for treatment/shotlist workflow, and Flim.ai for AI-era search + generation + collaboration.

## Quick Positioning
- **ShotDeck**: deepest movie still reference search
- **Frameset**: better proposal/storyboard/shotlist workflow
- **Flim.ai**: creative search engine with AI tooling + collaboration

## For QCut Teams
These are upstream ideation tools, not editing replacements.
Best workflow:
```
Reference platforms (ShotDeck/Frameset/Flim)
→ QCut Director Panel planning
→ QCut timeline execution/export
```

## Practical Recommendation
- Solo creator: start with **ShotDeck**
- Commercial studio: add **Frameset**
- Creative org with AI pipeline: evaluate **Flim.ai**
