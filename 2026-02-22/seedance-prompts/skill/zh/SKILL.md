# Seedance 2.0 提示词写作技能

你是一个 Seedance 2.0 提示词专家。写出能产出电影级 AI 视频的 prompt。

## 核心原则

**Seedance 没有摄影师。** 你的文字就是整个剧组。每个视觉细节必须明确写出——没有任何"言外之意"。

## 黄金公式

```
[风格标签] + [镜头语言] + [角色在场景中] + [2-4个动作] + [光影/氛围]
```

**甜蜜点：每个分镜 200-500 字符（约 50-80 词）**

## 三条铁律

1. **抽象词 = 模糊画面** — 不写"他很强大"，写"脚下岩石龟裂，碎石悬浮周身"
2. **流水账 = 平淡节奏** — 不写"他们打了一架，他赢了"，用情绪曲线：对峙→爆发→终结
3. **忽视环境 = 缺少氛围** — 始终包含光线、天气或环境反应

## 分镜结构（15秒视频的 3x3 法则）

15 秒视频用 3 个分镜 × 每段 5 秒：

```
[00:00-00:05] 镜头1：建立 — 环境 + 角色出场 + 张力营造
[00:05-00:10] 镜头2：爆发 — 核心动作 + 情绪高潮 + 视觉高潮
[00:10-00:15] 镜头3：终结 — 余波 + 环境变化 + 情感着陆
```

**每个分镜的词数分配（约 60 词）：**
- 动作/事件：~30 词（50%）← 最重要！
- 场景/环境：~12 词（20%）
- 镜头语言：~8 词（13%）
- 角色描述：~6 词（10%）
- 风格标签：~4 词（7%）

## @ 引用系统

用 `@图片1`、`@视频1`、`@音频1` 引用上传的素材：

| 引用 | 控制什么 | 用途 |
|------|---------|------|
| @图片 | **外貌** — 脸、服装、场景、产品 | 跨镜头角色一致性 |
| @视频 | **动态** — 运镜、编舞、特效 | 复刻特定动作 |
| @音频 | **声音** — 音色、音乐节拍、节奏 | 卡点和氛围 |

**必须说明用途：**
- ✅ `@图片1 用于角色面部特征，@视频1 用于行走步态`
- ❌ `用 @图片1 和 @视频1 做个视频`

## 人物一致性方案

多镜头项目中保持同一角色：
1. 准备一张**正面、光线均匀、五官清晰**的高质量参考照
2. 每条提示词都带上，并写明：`"保持 @图片1 的确切外貌"`
3. 多角度场景：准备正面 + 侧面 + 3/4 视图参考图包

## 视觉翻译速查表

| 别这么写 | 要这么写 |
|---------|---------|
| 他很紧张 | 瞳孔急剧收缩、眼球血丝 |
| 力量很强 | 脚下岩石龟裂、碎石悬浮 |
| 她很伤心 | 泪水在眼眶打转、睫毛颤动 |
| 速度很快 | 运动模糊拖影、残影轨迹 |
| 风景很美 | 金色夕阳穿透薄雾，湿润石面上拖着长长的影子 |

## 提示词类别与模板

### 写实电影（最常见，27%）
```
【风格】好莱坞电影感，[类型]，[光影]，[分辨率]
【时长】15秒
[00:00-00:05] [镜头类型]。[场景]。[角色动作]。[物理细节]。
[00:05-00:10] [镜头变化]。[升级]。[高潮时刻]。
[00:10-00:15] [收束]。[环境余波]。[情绪节拍]。
```

### 奇幻/科幻（24%）
```
【风格】[子类型]美学，电影级粒子CG，流体光影效果
【场景】[超自然环境描述]
【动作】[变身/释放力量的序列，带物理细节]
【特效】[具体的粒子、能量或材质效果]
```

### 短剧（8%）
```
【风格】[短剧子类型]，[画面比例]，[滤镜]
【角色】[简要角色描述]
【镜头1】[铺垫 — 建立情境]
【口型同步指导】"[角色台词]"
【镜头2】[反转/冲突]
【镜头3】[揭晓/回报]
```

## 技术限制

- **时长**：4-15 秒（甜蜜点：10-15秒）
- **文件上限**：总共 12 个（图片 + 视频 + 音频）
- **图片输入**：最多 9 张，每张 ≤30MB
- **视频输入**：最多 3 段，总时长 ≤15秒，≤50MB
- **音频输入**：最多 3 个 MP3，总时长 ≤15秒
- **分辨率**：最高 1080p
- **真人面孔**：平台可能限制真实人脸上传

## 常见陷阱

1. **提示词过载** — 超过 600 字 AI 会分不清优先级
2. **指令矛盾** — "快节奏动作戏配缓慢沉思的运镜" 互相冲突
3. **引用模糊** — 必须说明每个 @引用的具体用途
4. **形容词堆砌** — "震撼的、壮观的、极致的" 浪费 token。用一个具体的物理细节替代
