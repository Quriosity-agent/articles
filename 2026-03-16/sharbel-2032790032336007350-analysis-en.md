## Three Trends Worth Watching

### 1. Agent Infrastructure Dominates the List

At least 5 out of 10 projects directly serve building and running AI agents: OpenClaw (runtime), Superpowers (skills framework), Pi Mono (full-stack toolkit), Deer Flow (autonomous agent), and learn-claude-code (learning to build agents).

The signal is clear: **developers have moved from "using AI" to "building AI agents."** It's no longer about calling APIs — it's about building infrastructure that lets agents run continuously and work autonomously.

### 2. "No X Required" Is the New Selling Point

RuView's pitch: "no cameras, no sensors, just WiFi." AIRI's pitch: "self-hosted, you own it." learn-claude-code: "bash is all you need."

The pattern is interesting: **removing dependencies is itself innovation.** Every hardware requirement or cloud dependency you eliminate opens up a new cohort of users. For builders, this is a reliable design heuristic — ask "what can I remove?" before "what can I add?"

### 3. Chinese Teams Are Shipping Hard

MiroFish (swarm intelligence), Deer Flow (ByteDance), learn-claude-code (shareAI-lab) — Chinese developers are increasingly visible in the agent ecosystem. ByteDance's Deer Flow is particularly notable as a major tech company open-sourcing a SuperAgent framework.

## Practical Takeaways for Builders

**If you're building agent-related projects:**
- The OpenClaw + Superpowers combo is worth trying. One provides the runtime, the other provides plug-and-play skills. Don't reinvent the wheel.
- Pi Mono is another solid option — CLI + unified LLM API + Web UI + Slack bot in one package. Good for rapid agent prototyping.

**If you want to learn agent development:**
- learn-claude-code (9K stars) is a great starting point. Building a Claude Code clone from scratch with bash teaches you the core architecture of how agents actually work.
- claude-code-best-practice (11.8K stars) is your go-to reference for using Claude Code in production.

**If you're tracking the frontier:**
- RuView's WiFi-based pose detection sounds like science fiction, but it's a real and active research area. Worth watching if you're in IoT or smart home.
- Heretic (removing model guardrails) at #10 reflects persistent community demand for unlocked model capabilities. No judgment — just noting it's a real market signal.

## Bottom Line

Sharbel nailed it: **the pace of AI right now is insane.** Next month's list will look completely different. But the trends are clear — agent infrastructure, self-hosting, and lowering barriers to entry will keep accelerating.

If you're a builder, now is the best time. Not because everything is mature, but because everything is still changing. Get on the bus. You can fix the engine while it's moving.
