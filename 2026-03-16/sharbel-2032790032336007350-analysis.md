## 三个值得注意的趋势

### 1. Agent 基础设施占据半壁江山

10 个项目里，至少 5 个直接服务于 AI Agent 的构建和运行：OpenClaw（运行环境）、Superpowers（技能框架）、Pi Mono（全栈工具包）、Deer Flow（自主 Agent）、learn-claude-code（学习构建 Agent）。

这说明什么？**开发者已经从"用 AI"转向"造 AI Agent"。** 不只是调 API 了，而是在搭建让 Agent 持续运行、自主工作的基础设施。

### 2. "不需要 X" 成为新卖点

RuView 的卖点是"不需要摄像头、不需要传感器，只用 WiFi"。AIRI 的卖点是"自托管，数据归你"。learn-claude-code 说"bash is all you need"。

这个模式很有意思：**去除依赖本身就是创新。** 当你能用更少的东西做到同样的事，那就是降低门槛。对 builder 来说，每去掉一个硬件依赖或云服务依赖，就多了一批潜在用户。

### 3. 中国团队持续输出

MiroFish（群体智能）、Deer Flow（字节跳动）、learn-claude-code（shareAI-lab）——中国开发者在 Agent 领域的存在感越来越强。特别是字节的 Deer Flow，作为大厂开源的 SuperAgent 框架，值得关注。

## 对 Builder 的实操建议

**如果你在做 Agent 相关项目：**
- OpenClaw + Superpowers 的组合值得试。一个提供运行环境，一个提供即插即用技能，省得自己造轮子。
- Pi Mono 是另一个选择——CLI + LLM API + Web UI + Slack bot 一站式解决。适合想快速搭 Agent 原型的场景。

**如果你想学 Agent 开发：**
- learn-claude-code（9K stars）是个好起点。从零用 bash 构建 Claude Code 克隆，能帮你理解 Agent 的核心架构。
- claude-code-best-practice（11.8K stars）则是直接上手用 Claude Code 的最佳参考。

**如果你关注前沿应用：**
- RuView 用 WiFi 做姿态检测这件事，虽然听起来像黑科技，但确实是一个活跃的研究方向。如果你做 IoT 或智能家居，值得跟踪。
- Heretic（移除模型 guardrails）排第 10——说明社区对"解锁模型能力"的需求一直在。不评价对错，但这是真实需求。

## 一句话总结

Sharbel 说得对：**AI 的发展速度现在是疯狂的。** 下个月的榜单大概率完全不同。但趋势很清楚——Agent 基础设施、去中心化、降低门槛，这三条线会持续走强。

如果你是 builder，现在是最好的时候。不是因为一切都成熟了，而是因为一切都还在变。先上车，边开边修。
