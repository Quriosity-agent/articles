# Recordly: The Free Open-Source Screen Studio Alternative That Actually Works

> Source: [@LgyLight (三秋十李 Sergio)](https://x.com/lgylight/status/2033109870187295151) · 2026-03-15 · 60 likes

## TL;DR

Screen Studio is a popular macOS screen recorder known for auto-zoom, cursor animations, and polished backgrounds — perfect for product demos. But it costs $89. Now there are three open-source alternatives: **Openscreen** (the original), **Recordly** (recommended), and **CursorLens** (another fork). Verdict: **Recordly has the best features and can genuinely replace Screen Studio for simple demo recordings**.

---

## The Three Open-Source Contenders

### Openscreen — The Original

- First open-source Screen Studio alternative
- Solid foundation, but the maintainer doesn't want to make major architectural changes
- Website: [openscreen.vercel.app](https://openscreen.vercel.app)

### Recordly — Recommended ⭐

- Forked from Openscreen with substantial improvements
- **Key additions over Openscreen:**
  - Full cursor animation/rendering pipeline
  - Native macOS screen capture
  - Zoom animations faithful to Screen Studio
  - Smoother panning behavior
  - Drag-and-drop timeline editing interface
  - Save projects as `.recordly` files
- Cross-platform: macOS, Windows, and Linux
- Completely free and open-source
- GitHub: [github.com/webadderall/Recordly](https://github.com/webadderall/Recordly)
- Website: [recordly.dev](https://recordly.dev/)

### CursorLens — Another Fork

- Also forked from Openscreen
- Sits between Openscreen and Recordly in terms of features
- Fewer enhancements than Recordly

---

## Why Recordly Matters

1. **Cost** — Screen Studio is $89 per license. Recordly is free.
2. **Cross-platform** — Not macOS-only. Works on Windows and Linux too.
3. **Feature parity** — Auto-zoom, cursor animations, background styling — the core Screen Studio experience is there.
4. **Open source** — Don't like something? Change it.
5. **Active development** — The fork exists because there's real demand for features the original maintainer won't implement.

## Who Should Use This

- Developers recording product demos or tutorials
- Teams making changelog videos or feature showcases
- Anyone who wants Screen Studio polish without the price tag
- Windows/Linux users who couldn't use Screen Studio anyway

## Caveats

- The project itself warns "Expect bugs" — it's not 100% stable yet
- Polish gap vs Screen Studio still exists in edge cases
- Best for simple demo recordings; complex long-form video may still need professional tools
