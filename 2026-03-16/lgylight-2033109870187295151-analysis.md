# Recordly：免费开源的 Screen Studio 平替，录屏工具三兄弟横评

> 来源推文：[@LgyLight (三秋十李 Sergio)](https://x.com/lgylight/status/2033109870187295151) · 2026-03-15 · 60 likes

## TL;DR

Screen Studio 是 macOS 上很火的录屏工具，自动缩放、光标动画、好看的背景，录产品演示视频特别好用。但它收费不便宜。现在有三个开源替代品：**Openscreen**（鼻祖）、**Recordly**（推荐）、**CursorLens**（Fork）。结论：**Recordly 功能最强，可以直接平替 Screen Studio**。

---

## 三个开源项目对比

### Openscreen — 鼻祖

- 最早出现的开源 Screen Studio 替代品
- 基础功能齐全，但原作者不太愿意做大的架构改动
- 官网：[openscreen.vercel.app](https://openscreen.vercel.app)

### Recordly — 推荐 ⭐

- 从 Openscreen fork 出来，做了大量改进
- **新增功能：**
  - 完整的光标动画/渲染管线（cursor animation pipeline）
  - 原生 macOS 屏幕录制（native capture）
  - 忠实还原 Screen Studio 的缩放动画效果
  - 更平滑的画面平移行为
  - 拖拽式时间轴编辑界面
  - 项目文件保存为 `.recordly` 格式
- 支持 macOS、Windows、Linux
- 完全免费，代码开源
- GitHub：[github.com/webadderall/Recordly](https://github.com/webadderall/Recordly)
- 官网：[recordly.dev](https://recordly.dev/)

### CursorLens — 另一个 Fork

- 同样从 Openscreen fork 出来
- 功能介于 Openscreen 和 Recordly 之间
- 没有 Recordly 那么多改进

---

## 为什么 Recordly 值得关注

1. **省钱** — Screen Studio 单授权 $89，Recordly 免费
2. **跨平台** — 不只是 macOS，Windows 和 Linux 也能用
3. **功能接近** — 自动缩放、光标动画、背景美化，核心功能都有
4. **开源可定制** — 不满意可以自己改代码
5. **项目活跃** — 有 fork 的动力说明社区有需求，代码在迭代

## 适合谁用

- 需要录产品演示/教程视频的开发者
- 做 changelog 视频、feature showcase 的团队
- 想要 Screen Studio 效果但不想花钱的人
- 需要在 Windows/Linux 上录类似效果的人

## 注意事项

- 项目自己标注了 "Expect bugs"，不是 100% 稳定
- 和 Screen Studio 相比，细节打磨还有差距
- 适合录简单演示视频，复杂长视频可能还是需要专业工具
