## 这是什么

Varun Mathur 把 Karpathy 的 autoresearch loop 做成了通用平台。现在叫 **Hyperspace**，版本 v3.0.10。核心想法很简单：你用自然语言描述一个优化问题，网络自动起一群 agent 去解决，不用写代码。

三个新功能让这个系统从"有意思的实验"变成了"可能真有用的东西"。

## 三个核心功能

### 1. Autoswarms：开放进化计算网络

```bash
hyperspace swarm new "optimize CSS themes for WCAG accessibility contrast"
```

一行命令，系统自动做这些事：
- LLM 生成沙箱实验代码
- 本地多轮 dry-run 验证
- 发布到 P2P 网络，其他节点自动发现并加入
- 每个 agent 跑 **变异 → 评估 → 分享** 循环，全在 WASM 沙箱里
- 最好的策略自动传播
- Playbook curator 提炼出"为什么这个变异有效"，新加入的节点直接从积累的智慧起步

内置 3 个 swarm，任何人可以创建更多。

### 2. Research DAGs：跨领域复合智能

这是最有意思的部分。所有领域的实验都喂进一个共享的知识图谱（Research DAG）。

**实际效果：**
- 金融 agent 发现"砍掉弱因子 + 风险平价配置"能提升 Sharpe ratio
- 这个洞察自动传播到搜索 agent："也许砍掉低信号排序特征也能提升 NDCG？"
- ML agent 发现 RMSNorm 比 LayerNorm 好，skill-forging agent 就学到了归一化的 pattern

DAG 追踪每个领域的血缘链：`ml:★0.99←1.05←1.23 | search:★0.40←0.39 | finance:★1.32←1.24`

AutoThinker 循环读所有领域的数据，跨域综合、生成新假设。目前 DAG 有几百个节点，深度链到了 8+ 层。

### 3. Warps：自变异 Agent 配置

Warps 是声明式配置预设，改变你的 agent 在网络上做什么：

| Warp | 作用 |
|------|------|
| `enable-power-mode` | 最大化资源，全部功能开启 |
| `add-research-causes` | 开启所有领域的自动研究 |
| `optimize-inference` | 调优推理性能 |
| `privacy-mode` | 关闭遥测，本地推理 |
| `add-defi-research` | 启用 DeFi/加密研究 |
| `gpu-sentinel` | GPU 温控保护 |
| `enable-vault` | 本地加密 API key |

还能自然语言自定义：
```bash
hyperspace warp forge "enable cron job that backs up agent state to S3 every hour"
```

12 个内置 warp，社区 warp 通过 gossip 协议传播。可以叠加：`power-mode + add-research-causes + gpu-sentinel` = 把游戏 PC 变成自主研究站。

## 实际数据

237 个 agent 零人工干预做了什么：

| 领域 | Agent 数 | 实验次数 | 关键成果 |
|------|---------|---------|----------|
| ML 训练 | 116 | 728 | 验证 loss 降了 75%，一个 agent 发现 Kaiming 初始化后 23 个 peer 几小时内就采用了 |
| 搜索 | 170 | 21 策略 | BM25 调参、多样性惩罚、查询扩展，NDCG 从 0 到 0.40 |
| 金融 | 197 | 3,085 回测 | 自主收敛到"砍弱因子 + 风险平价"，Sharpe 1.32，3x 收益，5.5% 最大回撤 |
| 技能 | — | 3,795 | 本地 LLM 从头写 JS，异常检测/文本相似度/JSON diff 等 100% 正确率 |
| 基础设施 | 218 | 6,584 轮 | 自我优化网络本身 |

总计 14,832 次实验，跨 5 个领域。

## 对 Builder 的价值

**值得关注的点：**

1. **进化搜索 + 跨域传播的组合**是对的方向。单个发现不稀奇（CFA L2 就知道因子修剪），但 agent 自主收敛到正确结果说明方法论可行。

2. **WASM 沙箱 + P2P 传播**是工程上的好选择。安全隔离 + 去中心化发现，让分布式实验变得可行。

3. **Playbook curator 的设计**很聪明。不是只传播"好的结果"，而是传播"为什么好"。这让新节点不需要从零开始进化。

**需要注意的：**

- 目前的成果（Sharpe 1.32、NDCG 0.40）在专业标准下并不算顶级
- 真正的考验是这些 agent 跑几周甚至几个月后能不能持续产出新发现
- 跨域传播的假设（金融洞察 → 搜索优化）很有创意但未必总是有效

## 和 Karpathy 的 autoresearch 的关系

Varun 的引用推文提到，他们把 Karpathy 的 autoresearch 循环先用在了量化金融上（Autoquant v2.6.9）。135 个 agent 进化多因子交易策略，从 Sharpe ~1.04 的等权组合出发，自主收敛到 Sharpe 1.32——砍掉了红利、增长和趋势因子，切换到风险平价配置。

v3.0 把这个模式泛化成了任何人都能用的平台。

## 一句话总结

把进化计算 + 跨域知识图谱 + P2P 传播做成一个通用平台，让一群 agent 自己跑实验、自己发现规律、自己传播知识。方向对，但需要时间验证深度。
